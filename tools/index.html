Tools for control education

  • Bode Plot / Template — enter G(s) as text (parsed by math.js) to plot its frequency response, or turn the curve off for a blank Bode template (SVG)
  • Root Locus from Transfer Function — enter L(s) as text and draw its root locus with poles (×), zeros (○), asymptotes, breakaway points, and math coordinate labels (self-contained SVG)
